⚙️ Konfiguration krävs

För att spara pass behöver du koppla ett Google Formulär:

  1. Skapa ett Google Formulär med frågor: Datum, Meter, Anteckning
  2. Klicka på "Svar" → "Koppla till Kalkylark"
  3. Publicera kalkylarket: Arkiv → Dela → Publicera (CSV)
  4. Hämta formulärets entry-ID:n (se README)
  5. Uppdatera CONFIG i input.html
